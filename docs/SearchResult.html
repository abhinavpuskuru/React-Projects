Class: SearchResult

SearchResult(GET_GIT_TOPICS)

new SearchResult(GET_GIT_TOPICS)

Component responsible for querying GraphQL and retrieving Information to display on to the screen. searchItem => Has the Data String Entered in the Search Bar returns => Search Bar along with the data entered in the search bar

Parameters:
Name Type Description
GET_GIT_TOPICS string

The seachItem on the screen Displays all the "topics" related to the term "react" by default, using the GitHub GraphQL API along with the stargazers. A click on a topic will display the topics related to that topic along with the stargazers count.
