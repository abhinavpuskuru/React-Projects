1. Run the app

Now that the server is running, you can start the React app as well. The commands need to be run in a new terminal tab/window inside the root directory react-apollo

npm install
npm start

You can now open your browser and use the app on (http://localhost:3000). For Data calls Create .env file and Add REACT_APP_KEY = "Your Key"
